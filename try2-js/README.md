# Cuberoom vanila js

## Deploy

Run the command below

```sh
vc .
```

## Local test

`vc dev` can't deliver image files. Use sfz public to deliver static image
files.

```sh
vc dev

# in other shell
sfz public
```

## Add a domain to deploy

```sh
vercel alias xxx.vercel.app cuberoom-210404.juhyung.dev
```
